# 我的 vim 配置

## Overview
使用 Vundle 管理插件, .vimrc 和 rcfiles/ 里包含了一些特色配置.

## 安装

``` bash
git clone git@github.com:dodocat/dotvim.git .vim
cd .vim
ln -s ~/.vimrc ~/.vim/.vimrc
git submodule init
git submodule update
vim +BundleInstall +qa
```
